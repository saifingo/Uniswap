# Uinswap - Decentralized Exchange Mobile App

## Getting Started

### Prerequisites

- Node.js >= 14
- npm or yarn
- Expo CLI

### Installation

1. Clone the repository
```bash
git clone https://github.com/yourusername/uinswap.git
cd uinswap
```

2. Install dependencies
```bash
npm install
```

3. Start the development server
```bash
npx expo start
```
